Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Doplň vhodné odpovědi a využij k tomu nečíslovaného seznamu.

Použij v textu aspoň 3 internetové odkazy na zajímavé příklady využití JS!

  • JavaScript je programovací jazyk, který se používá hlavně pro tvorbu interaktivních prvků na webových stránkách, jako jsou animace, formuláře nebo dynamický obsah.
  • Lze jej využít na straně klienta (v prohlížeči) i na straně serveru (pomocí Node.js).
  • JavaScript umožňuje manipulaci s prvky v HTML pomocí DOM (Document Object Model) a může měnit vzhled webu bez nutnosti znovu načítat stránku.
  • Pro více informací a zajímavé příklady se podívejte na následující odkazy:
    • Interaktivní bloky
    • Simulace realného interaktivního ubrousku
    • Zajímavá šablona vytvořena JavaScriptem

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Use meaningful variable names
  2. Always use `let` or `const` instead of `var`
  3. Keep functions short and focused
  4. Comment your code for better understanding
  5. Avoid using global variables

3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>
// 2. Import JavaScriptu jako externí soubor
<script src="scripts.js"></script>
// 3. Použít DOM API pro manipulaci s prvky
document.getElementById('element').textContent = 'Hello World';

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // neukládá žádnou hodnotu
let x = 40 // celé číslo (typ number)
let pi = 3.14 // reálné číslo (typ number)
let exponent = 123e-2 // zápis desetinného čísla pomocí exponentu
let barva = "Modrá" // typ string
let prestupny = false // datový typ boolean, true or false

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, *=, /=, %= (zbytek po dělení)

Logické

&& (a), || (nebo), ! (negace)

Porovnávací

== (rovná se), === (rovná se a je stejného typu), != (nerovná se), !== (nerovná se a není stejného typu), >, <, >=, <=

Bitové

& (AND), | (OR), ^ (XOR), ~ (NOT), << (posun vlevo), >> (posun vpravo), >>> (posun vpravo s nulovým doplněním)

Typové

typeof (typ proměnné), instanceof (zjištění, zda objekt je)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Vyskočí okno v případě, že uživatel zadal špatné heslo. */
alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */
let password = prompt('Zadej heslo:');

confirm()

/* Potvrzení zdali opravdu chceme přejít na stránku seznam.cz */
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:

1. Pop-up okna mohou být rušivá a zhoršovat uživatelský zážitek.

2. Moderní prohlížeče často blokují pop-up okna, což omezuje jejich spolehlivost.
